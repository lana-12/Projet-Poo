# Tp PDO-POO


## Cloner le projet
```
git clone https://github.com/lana-12/Projet-Poo.git
```

## Dépendance
```
composer install
```

## Configuration
À changer dans :
```
Configuration/config.php
    - const DBHOST = serveur
    - const DBUSER = Identifiant
    - const DBPASS = Mot de passe
    - const DBNAME = Nom de la BDD

```

## Lancer le serveur

Commande :
```
php -S localhost:3000
```
